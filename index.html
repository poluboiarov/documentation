Редакторы

Копирование из документов

На данным момент всё настроено так, что при вставке текста из документа все лишние стили удаляются.

Все пустые строки между абзацами убираются, порядок заголовков проставляется соответственно документу, шаблонные стили для списков применяются автоматически.

Исключения!

«Чистка» не работает, если копировать текст из Google Docs, используя браузеры на базе Chromium: Google Chrome, Opera, Yandex browser.

Через браузер Mozilla Firefox и десктопные приложения копирование работает нормально.

Safari не проверено.

Подключение чанков

Список чанков и их параметры.

Для подключения чанка используется конструкция: [[$chunkName]].

Если чанк имеет входные параметры, то добавляем разделитель в виде знака вопроса и прописываем необходимые параметры и их значения: [[$chunkName? &title=`Заголовок` &ids=`23,24`]].

Осторожно!

Визуальный редактор "любит" вставлять ненужные тэги. Поэтому при размещение чанков желательно использовать вкладку "Источник" либо редактор кода, для того, чтобы избежать нежелательную разметку.

Получение динамической ссылки

Вместо абсолютной ссылки можно использовать сокращение: [[~id]].

Это удобнее, так как ссылка генерируется автоматически.

В будущем при необходимости сменить url страницы не нужно будет везде менять ссылку вручную.

Пример использования:

Вместо <a href="https://liber-vita.ru/narkologicheskaya-pomoshh/">
 Пишем <a href="[[~97]]"> (id ресурса в админке)

Плагины

Редактирование баннеров

Все баннеры находятся в разделе Пакеты -> BannerY.

Для создания нового баннера имеется соответствующая кнопка «Новый баннер».

Для редактирования уже имеющихся баннеров кликаем по нужному дважды ЛКМ или ПКМ -> «Изменить баннер».

Основные поля для заполнения:

  • Название: тип данных "text"
  • Текущее изображение: тип данных "image"
  • Текст баннера: тип данных "text/html"

Расположение заполненых полей на баннере соответственно:

Чанки

$doctors (врачи)

Карусель с выводом врачей.

Принимаемые параметры:

  • &title=`Заголовок секции` | (по умолчанию выводит "Наши врачи")
  • &ids=`Идентификаторы врачей` | (по умолчанию выводит всех врачей)

$reviews (отзывы)

Карусель с выводом отзывов.

Принимаемые параметры:

  • &title=`Заголовок секции` | (по умолчанию выводит "Отзывы")
  • &ids=`Идентификаторы отзывов` | (по умолчанию выводит все отзывы)

$blockquote (цитата)

Принимаемые параметры:

  • &text=`Текст цитаты` | (по умолчанию рыба-текст)
  • &name=`Имя цитирующего` | (Не обязательно! Если не заполнено, то выводится только текст цитаты)
  • &position=`Должность цитирующего` | (по умолчанию игнорируется, выводится только если прописано имя)
  • &image=`Ссылка на изображение цитирующего` | (по умолчанию тестовое изображение, выводится только если прописано имя)
    Ссылка на файл строится от корня сайта, например:
    assets/images/users/user-betty-wade-60x60.jpg

$certifications (лицензии)

Карусель с выводом лицензий.

Нет принимаемых параметров.

$prices (цены)

Вывод секции с ценами до трех блоков.
Работает только в шаблоне услуг.

Принимаемые параметры:

  • &title=`Заголовок секции` | (по умолчанию выводит "Цены на услуги")

Наследование

Редактор с ценами находится именно у родительской категории! Вложенные услуги наследуют цены у верхнего уровня.

$recommendation (подтвержденная информация)

Блок "Доктор проверил".

Принимаемые параметры:

  • &name=`Имя врача` | (по умолчанию выводит "Кристина Ивановна")
  • &position=`Специальность врача` | (по умолчанию игнорируется)
  • &text=`Сообщение от врача` | (по умолчанию рыба-текст)
  • &image=`Ссылка на изображение врача` | (по умолчанию тестовое изображение)
    Ссылка на файл строится от корня сайта, например:
    assets/images/our-doctors/doctor.jpg

$form_phone (форма с телефоном)

Форма с полем для ввода телефона. Так же отображает основной телефон клиники, как альтернатива вводу телефона.

Принимаемые параметры:

  • &title=`Заголовок формы` | (по умолчанию выводит "Хотите узнать, как вылечить алкоголизм или наркоманию?")
  • &text=`Описание формы` | (по умолчанию выводи "Оставьте ваш номер
    и получите бесплатную консультацию")
  • &button=`Текст кнопки` | (по умолчанию выводит "Позвонить мне")

$equipage (перечень городов Подмосковья)

Блок с выводом колонок городов Подмосковья с якорями на блок "Почему стоит выбрать клинику Либер Вита"

Не принимает параметры.
